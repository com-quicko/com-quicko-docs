Quicko Tools provides a thin, lightweight SDK for native mobile applications. The SDK uses Material 3 design principles to blend seamlessly with your app's existing theme.

## Overview

The mobile SDK enables you to embed Quicko Tools natively within your iOS or Android application, providing a smooth user experience that feels part of your app.

<AccordionGroup>
  <Accordion title="Lightweight" icon="feather" defaultOpen>

  </Accordion>
  <Accordion title="Native Experience" icon="smartphone" defaultOpen>
    Tools render natively within your app, providing smooth animations and interactions.
  </Accordion>
  <Accordion title="Material 3 Design" icon="palette">
    The SDK follows Material 3 specifications, automatically adapting to your app's theme.
  </Accordion>
</AccordionGroup>

## Installation

<Tabs>
  <Tab title="iOS (Swift)">
    Add the QuickoTools SDK to your project using Swift Package Manager:

    ```swift
    dependencies: [
        .package(url: "https://github.com/nicq-io/quicko-tools-ios.git", from: "1.0.0")
    ]
    ```

    Or add via Xcode:
    1. Go to **File â†’ Add Packages**
    2. Enter the repository URL
    3. Select the version and add to your target
  </Tab>
  <Tab title="Android (Kotlin)">
    Add the dependency to your `build.gradle`:

    ```kotlin
    dependencies {
        implementation("com.nicq.tools:quicko-tools:1.0.0")
    }
    ```

    Make sure you have the Maven Central repository in your project:

    ```kotlin
    repositories {
        mavenCentral()
    }
    ```
  </Tab>
</Tabs>

## Quick Start

<Tabs>
  <Tab title="iOS (Swift)">
    ```swift
    import QuickoTools

    // Create options with theme configuration
    let theme: [String: Any] = [
        "mode": "light",
        "seed": "#006AD4"
    ]

    // Present a tool
    QuickoTools.launchHraCalculator(theme: theme, from: self)
    ```
  </Tab>
  <Tab title="Android (Kotlin)">
    ```kotlin
    import com.nicq.tools.QuickoTools
    import org.json.JSONObject

    // Create options with theme configuration
    val theme = JSONObject().apply {
        put("mode", "light")
        put("seed", "#006AD4")
    }

    // Launch a tool
    QuickoTools.launchHraCalculator(this, theme)
    ```
  </Tab>
</Tabs>

## Options Structure

The SDK accepts an options object to customize the tool's appearance:

```json
"theme": {
    "mode": "light",
    "seed": "#006AD4"
}
```

| Property | Type | Description |
|----------|------|-------------|
| `theme.mode` | `String` | Theme mode: `light` or `dark` |
| `theme.seed` | `String` | Seed color (hex with #) for theme generation |

## Integration Steps

<Steps>
  <Step title="Install the SDK">
    Add the QuickoTools dependency to your iOS or Android project.
  </Step>
  <Step title="Configure Options">
    Create an theme object with your preferred mode and brand color.
  </Step>
  <Step title="Present the Tool">
    Use the SDK's launch methods to present the desired tool to your users.
  </Step>
  <Step title="Handle Results">
    Optionally, implement delegate/callback methods to receive results or events from the tool.
  </Step>
</Steps>
