## Employer Details

| Serial No. (Field) | Param      | Type   | Required | Description                                                      |
| ------------------ | ---------- | ------ | -------- | ---------------------------------------------------------------- |
| 1.                 | Name       | string | Yes      | Name of the employer                                              |
| 2.                 | TAN        | string | Yes      | Tax Deduction and Collection Account Number                      |
| 3.                 | PAN        | string | Yes      | Permanent Account Number of the employer                         |
| 4.                 | Address    | string | Yes      | Address of the workplace of the employer                         |
| 5.                 | City       | string | Yes      | Address of the workplace of the employer (City)                  |
| 6.                 | State      | string | Yes      | Address of the workplace of the employer (State)enums: State outside India, Assam, Gujarat, etc. |
| 7.                 | Postal Code | number | Yes      | Address of the workplace of the employer (Postal Code)           |
| 8.                 | Country    | string | Yes      | Country where employer belongs                                   |
